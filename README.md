# Caminex El Salvador - Alojamientos Turísticos

Proyecto de landing page interactiva desarrollada para mostrar alojamientos turísticos en El Salvador.

##  URL del Sitio Publicado
Visita el proyecto en vivo aquí: https://teranorellana.github.io/caminex-alojamientos-sv/

##  Tecnologías Utilizadas
* HTML5
* CSS3 (Grid y Flexbox)
* JavaScript (Vanilla JS)
* Control de versiones con Git/GitHub
* Despliegue en GitHub Pages

##  Funcionalidades
* Diseño completamente responsive.
* Menú de navegación fijo con menú hamburguesa para dispositivos móviles.
* Tarjetas de servicios maquetadas con CSS Grid.
* Formulario de contacto validado con JavaScript (prevención de envíos vacíos y validación de formato de correo).
* Manipulación dinámica del DOM para dar feedback de éxito/error al usuario.
